relay는 **Claude Code 플러그인**입니다. 명령어 하나로 모든 것이 설치됩니다: MCP 서버(메시지 버스, 태스크 보드, 메모리 레이어), `/relay:relay`, `/relay:init`, `/relay:agent` Skills, 그리고 실시간 대시보드에 상태를 전송하는 PostToolUse 훅까지.

## 사전 요구사항

- [Node.js](https://nodejs.org) v18 이상
- [Claude Code](https://claude.ai/download) 설치 및 로그인 완료

## 설치

<Tabs>
  <TabItem label="전역 설치 (권장)">
    모든 프로젝트에서 사용 가능합니다:

    ```bash
    claude mcp add --scope user relay -- npx -y --package @custardcream/relay relay
    ```

    Skills는 첫 로드 시 `~/.claude/skills/`에 자동 설치됩니다.
  </TabItem>
  <TabItem label="로컬 설치 (현재 프로젝트만)">
    현재 프로젝트에만 적용됩니다:

    ```bash
    claude mcp add --scope local relay -- npx -y --package @custardcream/relay relay
    ```

    Skills는 프로젝트 내 `.claude/skills/`에 설치됩니다. 전역과 로컬이 모두 설치된 경우 로컬이 우선 적용됩니다.
  </TabItem>
</Tabs>

`--`는 `claude mcp add` 플래그와 relay 서버 명령어를 구분합니다. `--` 이후의 모든 내용이 Claude Code가 MCP 서버를 시작할 때 실행하는 명령어입니다.

## 설치 확인

Claude Code를 재시작한 후:

```
/relay:init
```

Skills가 로드되고 MCP 서버가 연결되면 설치가 완료된 것입니다.
