relay is distributed as an npm package. Register the MCP server with Claude Code using the command below.

## Prerequisites

- [Node.js](https://nodejs.org) v18 or later (or any Node-compatible runtime)
- [Claude Code](https://claude.ai/download) installed and signed in

## Installation

<Tabs>
  <TabItem label="Global (recommended)">
    Register the relay MCP server globally so it is available in all projects:

    ```bash
    claude mcp add --scope user relay -- npx -y --package @custardcream/relay relay
    ```

    Skills and hooks are installed automatically when the plugin is loaded.
  </TabItem>
  <TabItem label="Local (current project only)">
    To scope relay to a single project, use `--scope local` instead:

    ```bash
    claude mcp add --scope local relay -- npx -y --package @custardcream/relay relay
    ```

    Local installation overrides global when both are present.
  </TabItem>
</Tabs>

## Verify installation

After restarting Claude Code, confirm the installation with:

```
/relay:init
```

If the relay MCP server connects successfully, the installation is complete.
