relay is a **Claude Code plugin**. One command installs everything: the MCP server (message bus, task board, memory layer), the `/relay:relay`, `/relay:init`, and `/relay:agent` skills, and the PostToolUse hook that feeds the real-time dashboard.

## Prerequisites

- [Node.js](https://nodejs.org) v18 or later
- [Claude Code](https://claude.ai/download) installed and authenticated

## Install

<Tabs>
  <TabItem label="Global (recommended)">
    Available in all your projects:

    ```bash
    claude mcp add --scope user relay -- npx -y --package @custardcream/relay relay
    ```

    Skills are installed to `~/.claude/skills/` automatically on first load.
  </TabItem>
  <TabItem label="Local (current project only)">
    Scoped to the current project only:

    ```bash
    claude mcp add --scope local relay -- npx -y --package @custardcream/relay relay
    ```

    Skills are installed to `.claude/skills/` inside the project. Local overrides global when both are present.
  </TabItem>
</Tabs>

The `--` separates `claude mcp add` flags from the relay server command. Everything after `--` is the command Claude Code runs to start the MCP server.

## Verify

Restart Claude Code, then run:

```
/relay:init
```

If the skills load and the MCP server connects, installation is complete.
