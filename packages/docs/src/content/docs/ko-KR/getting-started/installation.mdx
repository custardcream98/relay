relay는 npm 패키지로 배포됩니다. 아래 명령어로 Claude Code에 MCP 서버를 등록하세요.

## 사전 요구사항

- [Node.js](https://nodejs.org) v18 이상 (또는 Node 호환 런타임)
- [Claude Code](https://claude.ai/download) 설치 및 로그인 완료

## 설치

<Tabs>
  <TabItem label="전역 설치 (권장)">
    모든 프로젝트에서 사용할 수 있도록 relay MCP 서버를 전역으로 등록합니다:

    ```bash
    claude mcp add --scope user relay -- npx -y --package @custardcream/relay relay
    ```

    플러그인이 로드될 때 Skills와 Hooks가 자동으로 설치됩니다.
  </TabItem>
  <TabItem label="로컬 설치 (현재 프로젝트만)">
    특정 프로젝트에만 relay를 적용하려면 `--scope local`을 사용하세요:

    ```bash
    claude mcp add --scope local relay -- npx -y --package @custardcream/relay relay
    ```

    전역과 로컬이 모두 설치된 경우 로컬이 우선 적용됩니다.
  </TabItem>
</Tabs>

## 설치 확인

Claude Code를 재시작한 후, 다음 명령으로 설치를 확인하세요:

```
/relay:init
```

relay MCP 서버가 정상적으로 연결되면 설치가 완료된 것입니다.
