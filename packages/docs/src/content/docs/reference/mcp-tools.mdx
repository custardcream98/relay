---
title: MCP 툴 레퍼런스
description: relay MCP 서버가 제공하는 모든 툴의 파라미터와 반환값
---

모든 툴은 `{ success: boolean, data?, error? }` 형태로 응답한다.
파라미터에 `agent_id`가 항상 포함되어 누가 호출하는지 추적한다.

## 메시징

### send_message

에이전트 간 메시지를 전송한다.

**파라미터:**
```typescript
{
  agent_id: string;   // 발신 에이전트 ID
  to: string | null;  // 수신 에이전트 ID. null이면 브로드캐스트
  content: string;    // 메시지 내용
  session_id: string; // 세션 ID
}
```

**반환:**
```typescript
{
  success: true;
  message_id: string;
}
```

---

### get_messages

수신된 메시지를 조회한다.

**파라미터:**
```typescript
{
  agent_id: string;   // 조회할 에이전트 ID
  session_id: string; // 세션 ID
  since?: string;     // 이 시간 이후의 메시지만 조회 (ISO 8601)
}
```

**반환:**
```typescript
{
  success: true;
  messages: Array<{
    id: string;
    from: string;
    to: string | null;
    content: string;
    created_at: string;
  }>;
}
```

## 태스크

### create_task

새 이슈/태스크를 생성한다.

**파라미터:**
```typescript
{
  agent_id: string;        // 생성 에이전트 ID
  session_id: string;      // 세션 ID
  title: string;           // 태스크 제목
  description?: string;    // 완료 기준이 명확한 상세 설명
  assignee: string;        // 담당 에이전트 ID
  priority: "critical" | "high" | "medium" | "low";
}
```

**반환:**
```typescript
{
  success: true;
  task_id: string;
}
```

---

### update_task

태스크 상태 또는 담당자를 변경한다.

**파라미터:**
```typescript
{
  agent_id: string;    // 호출 에이전트 ID
  session_id: string;  // 세션 ID
  task_id: string;     // 태스크 ID
  status?: "todo" | "in_progress" | "done";
  assignee?: string;   // 담당자 변경 시
}
```

---

### get_my_tasks

내 담당 태스크를 조회한다.

**파라미터:**
```typescript
{
  agent_id: string;   // 조회할 에이전트 ID
  session_id: string; // 세션 ID
}
```

**반환:**
```typescript
{
  success: true;
  tasks: Array<{
    id: string;
    title: string;
    description: string | null;
    status: string;
    priority: string;
    assignee: string;
    created_at: string;
  }>;
}
```

## 아티팩트

### post_artifact

산출물(디자인 스펙, PR, 리포트 등)을 공유한다.

**파라미터:**
```typescript
{
  agent_id: string;    // 공유 에이전트 ID
  session_id: string;  // 세션 ID
  name: string;        // 아티팩트 이름 (예: "login-fe-pr")
  type: string;        // 타입 (예: "pr", "figma_spec", "analytics_plan", "report")
  content: string;     // 아티팩트 내용 (텍스트 또는 JSON 문자열)
}
```

**반환:**
```typescript
{
  success: true;
  artifact_id: string;
}
```

---

### get_artifact

아티팩트를 조회한다.

**파라미터:**
```typescript
{
  agent_id: string;    // 조회 에이전트 ID
  session_id: string;  // 세션 ID
  name: string;        // 아티팩트 이름
}
```

## 리뷰

### request_review

코드 리뷰를 요청한다.

**파라미터:**
```typescript
{
  agent_id: string;    // 요청 에이전트 ID
  session_id: string;  // 세션 ID
  artifact_id: string; // 리뷰 대상 아티팩트 ID
  reviewer: string;    // 리뷰어 에이전트 ID
}
```

**반환:**
```typescript
{
  success: true;
  review_id: string;
}
```

---

### submit_review

리뷰 결과를 제출한다.

**파라미터:**
```typescript
{
  agent_id: string;            // 리뷰어 에이전트 ID
  session_id: string;          // 세션 ID
  review_id: string;           // 리뷰 ID
  status: "approved" | "changes_requested";
  comments?: string;           // 리뷰 코멘트
}
```

## 메모리

### read_memory

에이전트의 기억(또는 프로젝트 기억)을 읽는다.

**파라미터:**
```typescript
{
  agent_id?: string;  // 생략하면 project.md + lessons.md 반환
}
```

**반환:**
```typescript
{
  success: true;
  content: string;    // 마크다운 텍스트
}
```

---

### write_memory

에이전트 기억의 특정 섹션을 갱신한다.

**파라미터:**
```typescript
{
  agent_id: string; // 에이전트 ID
  key: string;      // 섹션 키 (예: "tech-stack", "conventions")
  content: string;  // 새 내용
}
```

---

### append_memory

에이전트 기억에 내용을 누적 추가한다.

**파라미터:**
```typescript
{
  agent_id?: string; // 생략하면 lessons.md에 추가
  content: string;   // 추가할 내용
}
```

## 세션

### list_sessions

과거 세션 목록을 조회한다.

**반환:**
```typescript
{
  success: true;
  sessions: Array<{
    id: string;
    created_at: string;
    summary: string | null;
  }>;
}
```

---

### get_session_summary

특정 세션의 요약을 조회한다.

**파라미터:**
```typescript
{
  session_id: string; // 세션 ID
}
```

---

### list_agents

사용 가능한 에이전트 목록을 조회한다.

**반환:**
```typescript
{
  success: true;
  agents: Array<{
    id: string;
    name: string;
    description: string;
    systemPrompt: string;
    tools: string[];
  }>;
}
```
