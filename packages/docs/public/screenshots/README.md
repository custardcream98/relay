# Screenshots

Place actual relay dashboard screenshots here.

## How to capture

1. Run `relay` in a project with an agent pool configured.
2. Start a session: `/relay:relay "your task"`
3. Open `http://localhost:3456` in a browser while agents are running.
4. Capture a screenshot at **1280×800** resolution.

## File naming convention

| File | Description |
|---|---|
| `dashboard.png` | Full 3-panel dashboard view |
| `task-board.png` | Task Board (Kanban) panel only |
| `agent-thoughts.png` | Agent Thoughts panel only |
| `messages.png` | Message Feed panel only |
